# Atalaya Flutter Core Source

Este paquete contiene el codigo fuente principal de Flutter que se pudo reconstruir.

Incluye:
- lib/
- pubspec.yaml
- analysis_options.yaml
- docs/api_contract.md
- test/unit_converter_test.dart

No incluye carpetas generadas por Flutter para plataformas.
Usa el paquete de scripts de regeneracion para recrearlas con `flutter create .`.
